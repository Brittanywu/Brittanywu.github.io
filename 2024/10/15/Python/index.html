File IO

write a file

file opening modes

Modes Description
r open a file for reading only, pointer at beginning
w open a file for writing only.
if exist, overwrite
if not exist, create a new file for writing
a open a file for appending
if exist, pointer at the end
if not exist, create a new file
_b in binary format
__+ open a file for both writing an d reading

python escape characher

Code Result Example
\‘ Single Quote 'It\'s a sunny day.'，在字符串中插入单引号 '。
\\ Backslash "C:\\Program Files\\MyApp"，表示文件路径中的反斜杠
==\n== New Line "Hello\nWorld!"
\r Carriage Return 将光标移动到当前行的开头，但不换行。(覆盖\r之前的输入)
==\t== Tab "Name\tAge\tLocation"，会产生列对齐的效果。
\b Backspace "Helloo\b"，输出为 Hello
\f Form Feed 换页符
\ooo Octal value \141 表示小写字母 a，因为 141 是字母 a 的八进制 ASCII 值。
\xhh Hex value \x61 表示小写字母 a，因为 61 是字母 a 的十六进制 ASCII 值。
