704. Binary Search

二分查找两种写法,按不同区间来区分

  1. target -> [left,right]

  2. target -> [left,right)

  • 定义right：1. len(nums)-1 2.len(nums)

易错点

切换指针时候: left=middle+1 / right=middle-1
