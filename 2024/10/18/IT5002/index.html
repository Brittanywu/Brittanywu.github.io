signed/unsigned numbers

  • Unsigned numbers:只有正数和0

  • Signed numbers:包括正数和负数

Sign-and-Magnitude

negative: invert the sign

  • 0 for positive, 1 for negative -> 0 100 1100 (8 bits) 0 for sign/7 for magnitude

  • largest number: 0111 1111=+127 smallest number: 1111 1111=-127

  • two zreo:(+0) 0000 0000/(-0)1000 0000

1s Complement -x=2n-x-1

negative: inverte all bits

  • largest:0111 1111=+127 **smallest:**1000 0000=-127

  • two zeros:(+0)0000 0000/(-0)1111 1111

2s Complement -x=2n-x

negate:

  1. invert all bits, and add 1 in bits
  2. 从右到左保留第一个1和后面所有0，前面的bit都反过来

  • **largest:**0111 1111=+127. **smallest:**1000 0000=-128

  • one zero:(+0)0000 0000

    • 没有-0

compare

1s 2s
取值范围 -(2^(n-1)-1) ~ 2^(n-1)-1 -2^(n-1) ~ 2^(n-1)-1 因为2s没有-0，所以负数位要多一位
减法 ignore the carry out of MSB end-around carry

Decision-making instructions

conditional Branch

beq $r1,$r2,L1 beq: branch if equal

  • go to label L1 if value in r1=r2

bne $r1,$r2,L1 bne: branch if not equal

  • go to label L1 if value in r1!=r2

unconditional branch

j L1 jump to label L1 unconditionally

inequalities

赋值：slt $t0,$s1,$s2 set on less than:

  • if (s1<s2): t0=1; else t0=0

小于：blt $s1,$s2,L branch if less than:

  • if (s1<s2) go to L
  • slt $t0,$s1,$s2 bne $t0,$zero,L

大于等于：bge $t1,$t2,L branch if great than or equal

  • if(t1>=t2) go to L
